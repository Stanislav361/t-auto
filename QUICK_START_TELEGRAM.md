# 🚀 Быстрый старт: Telegram бот для заявок

## ⚡ За 5 минут

### 1️⃣ Создайте бота (2 минуты)
1. Откройте Telegram → найдите **@BotFather**
2. Отправьте: `/newbot`
3. Введите имя: `T-AUTO Заявки`
4. Введите username: `t_auto_zayavki_bot` (должен заканчиваться на `bot`)
5. **Скопируйте токен** (выглядит как: `123456789:ABCdef...`)

### 2️⃣ Получите Chat ID (1 минута)
1. Найдите **@userinfobot** в Telegram
2. Отправьте `/start`
3. **Скопируйте ваш Chat ID** (число, например: `123456789`)

### 3️⃣ Настройте проект (1 минута)
1. Создайте файл `.env.local` в папке `111`
2. Добавьте:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
TELEGRAM_CHAT_ID=ваш_chat_id
```

### 4️⃣ Проверьте (1 минута)
1. Запустите: `npm run dev`
2. Откройте: `http://localhost:3000/contacts`
3. Заполните форму и отправьте
4. Проверьте Telegram - должно прийти сообщение! ✅

## 📝 Пример .env.local

```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=123456789
```

## ❓ Проблемы?

- **Бот не отвечает?** Проверьте токен
- **Сообщения не приходят?** Проверьте Chat ID
- **Ошибка 401?** Токен неправильный
- **Ошибка 400?** Chat ID неправильный

Подробная инструкция: `TELEGRAM_BOT_SETUP.md`
