# Инструкция по настройке Telegram бота для получения заявок

## 📋 Пошаговая инструкция

### Шаг 1: Создание бота через BotFather

1. **Откройте Telegram** на телефоне или компьютере
2. **Найдите бота @BotFather** в поиске Telegram
3. **Отправьте команду:** `/start`
4. **Отправьте команду:** `/newbot`
5. **Введите имя бота** (например: `T-AUTO Заявки`)
6. **Введите username бота** (должен заканчиваться на `bot`, например: `t_auto_zayavki_bot`)
7. **Скопируйте токен**, который выдаст BotFather (выглядит так: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

> ⚠️ **ВАЖНО:** Сохраните токен в безопасном месте! Он нужен для работы бота.

### Шаг 2: Получение Chat ID (ID вашего чата)

Есть два способа:

#### Способ 1: Через бота @userinfobot
1. Найдите бота `@userinfobot` в Telegram
2. Отправьте ему `/start`
3. Он покажет ваш Chat ID (например: `123456789`)

#### Способ 2: Через бота @getidsbot
1. Найдите бота `@getidsbot` в Telegram
2. Отправьте ему любое сообщение
3. Он покажет ваш Chat ID

> 💡 **Совет:** Если хотите получать заявки в группу, добавьте бота в группу и используйте ID группы.

### Шаг 3: Настройка переменных окружения

1. Создайте файл `.env.local` в корне проекта (папка `111`)
2. Добавьте туда:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
TELEGRAM_CHAT_ID=ваш_chat_id
```

**Пример:**
```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=123456789
```

> ⚠️ **ВАЖНО:** 
> - Не коммитьте файл `.env.local` в Git!
> - Этот файл уже добавлен в `.gitignore`

### Шаг 4: Проверка работы

1. Запустите сайт: `npm run dev`
2. Откройте страницу контактов: `http://localhost:3000/contacts`
3. Заполните форму и отправьте заявку
4. Проверьте Telegram - должно прийти сообщение с заявкой

## 🔧 Дополнительные настройки

### Настройка бота для группы

Если хотите получать заявки в группу:

1. Создайте группу в Telegram
2. Добавьте вашего бота в группу (через BotFather: `/setjoingroups` → выберите бота → `Enable`)
3. Добавьте бота как администратора группы
4. Получите ID группы:
   - Добавьте бота `@getidsbot` в группу
   - Отправьте любое сообщение в группу
   - Бот покажет ID группы (начинается с `-`, например: `-1001234567890`)
5. Используйте этот ID в `.env.local`

### Настройка уведомлений

Бот будет отправлять сообщения в формате:
```
🆕 Новая заявка с сайта

👤 Имя: [Имя клиента]
📞 Телефон: [Телефон]
💬 Сообщение: [Текст сообщения]

⏰ Время: [Дата и время]
```

## 🚨 Решение проблем

### Бот не отвечает
- Проверьте правильность токена
- Убедитесь, что бот запущен (отправьте `/start` боту)

### Сообщения не приходят
- Проверьте Chat ID (должен быть числом)
- Убедитесь, что бот добавлен в группу (если используете группу)
- Проверьте консоль браузера на ошибки

### Ошибка 401 Unauthorized
- Токен неправильный или истек
- Создайте нового бота через BotFather

### Ошибка 400 Bad Request
- Chat ID неправильный
- Проверьте формат ID (число, для групп начинается с `-`)

## 📞 Поддержка

Если возникли проблемы:
1. Проверьте консоль браузера (F12)
2. Проверьте логи сервера
3. Убедитесь, что переменные окружения установлены правильно
