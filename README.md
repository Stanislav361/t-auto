# E.N.SERVICE Website

Этот проект был создан на основе фронтенда из папки `frontend`, но переработан в полноценный веб-сайт.

## Установка и запуск

1. **Важно:** Скопируйте папку `public` из `frontend` в эту папку (`111`). Это нужно для работы картинок и звуков.
   - `frontend/public` -> `111/public`

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

4. Откройте http://localhost:3000

## Структура

- `app/page.tsx`: Главная страница со Splash Screen и лендингом.
- `app/services/`: Страница услуг.
- `app/about/`: О компании.
- `app/portfolio/`: Наши работы.
- `app/contacts/`: Контакты.
- `components/`: Компоненты UI, Header, Footer.

## Цветовая схема

Используется черно-желтая гамма:
- Primary: Yellow (#FFCC00)
- Background: Black (#000000)
